Blog

En este post cuento cómo empecé en el desarrollo web y mis primeros proyectos.

Durante el proceso, aprendí la importancia de escribir código limpio y bien organizado, así como el uso de etiquetas semánticas para mejorar la estructura del contenido. También descubrí cómo utilizar propiedades de CSS para mejorar el diseño visual, como colores, márgenes, tipografías y distribución de elementos.

Explico el proceso de crear un CV web utilizando HTML y CSS.

Me enfoqué en organizar correctamente los apartados más importantes como datos personales, experiencia, educación y habilidades, buscando que la información fuera clara y fácil de leer para cualquier persona.
También trabajé en el diseño responsive para que el CV se visualice correctamente en diferentes dispositivos, como móviles o tablets, lo cual es fundamental hoy en día. Este proyecto me permitió aplicar conocimientos prácticos y mejorar mi capacidad de maquetación web.

Introducción al backend y cómo conectar con bases de datos usando PHP.

Aprendí a crear formularios, procesar datos enviados por el usuario y almacenarlos en una base de datos. También trabajé con consultas SQL básicas para insertar, consultar y gestionar información.
Este aprendizaje fue clave para comprender la conexión entre el frontend y el backend, y cómo se pueden desarrollar aplicaciones más completas que interactúan con el usuario y almacenan información de manera eficiente.
